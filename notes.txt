EDA REMINDER NOTES (Quick Reference)


1️⃣ Missing Values Check
 If count == total rows → ✅ No missing values
 If count < total rows → ❌ Missing values present

2️⃣ Spread Check (Standard Deviation)
  std < mean  → Values are close (stable feature)
  std ≈ mean  → Medium spread
  std > mean  → High spread, possible outliers



3️⃣ Skewness Check (Mean vs Median)
  mean ≈ median  → Normal distribution
  mean > median  → Rightskewed data
  mean < median  → Leftskewed data



4️⃣ Outlier Detection Hint
  max ≈ Q3  → No major outliers
  max >> Q3  → Upper outliers present
  min << Q1  → Lower outliers present



 5️⃣ Quartile Concept
 **Q1 (25%)** → 25% of data lies below this value
 **Q2 (50%) / Median** → Middle value (50% below, 50% above)
 **Q3 (75%)** → 75% of data lies below this value




Correlation measures the strength and direction of a linear relationship between features, helping identify predictors of during EDA
